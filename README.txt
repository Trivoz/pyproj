				     PyProj

PyProj is a tool to for managing Python projects. It has been designed as an
alternative to default Python packaging tools, such as `setuptools` and `distutils`.

PyProj is currently in development, and is not yet ready for use, however you can
still install it using cargo (Rust's package manager) by running cargo install pyproj.
